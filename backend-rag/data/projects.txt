Project: Computer Vision-Based Driver Drowsiness Detector
Role: Machine Learning Engineer
Technologies: Python, OpenCV, MobileNet, Deep Learning
Description:
Developed a real-time driver drowsiness detection system using computer vision
and deep learning techniques to improve road safety.

Approach:
Used MobileNet with transfer learning for eye-state classification.
Integrated webcam-based real-time monitoring and applied threshold-based logic
to detect prolonged eye closure.

Outcome:
Achieved 91% accuracy in detecting driver drowsiness.
Implemented an audio alert mechanism to warn drivers in real time.
